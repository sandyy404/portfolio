# Sandeep Kumar Pandey - Developer Portfolio Website

A modern, high-performance, award-winning developer portfolio built for **Sandeep Kumar Pandey** (B.Tech Computer Science Engineering student at Galgotias University, Backend & AI Developer).

![Portfolio Preview](https://images.unsplash.com/photo-1618005182384-a83a8bd57fbe?auto=format&fit=crop&w=1200&q=80)

## 🚀 Key Features

- **3D Interactive Hero Canvas**: WebGL particle constellation rendered with Three.js reacting to real-time cursor movements.
- **Glassmorphism & Cyber Blue Theme**: Custom HSL dark blue (`#030712`), cyan glow accents (`#00F0FF`), and backdrop blur cards.
- **Dynamic Command Palette (`Ctrl + K`)**: Instant keyboard search for navigation, resume download, AI agent launch, and social profiles.
- **Embedded AI Portfolio Assistant**: Floating natural-language chat agent answering questions about Sandeep's experience, Spring Boot backend skills, and Galgotias University education.
- **Interactive Tech Stack Visualizer**: Categorized skill grid (Java, Spring Boot, MySQL, React, Python, REST APIs) with animated level meters.
- **Featured Projects Showcase**: Detailed cards for *Volunteer Management System*, *Employee Payroll System*, *Online Shopping Cart*, and *Student Record Management System* with architecture modals.
- **GitHub Contribution Heatmap**: Visual contribution activity grid reflecting open-source engagement.
- **SEO & Accessibility**: Complete meta tags, OpenGraph tags, sitemap.xml, robots.txt, semantic HTML5, and WCAG AA color contrast compliance.

---

## 🛠️ Technology Stack

- **Frontend Core**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS v3, Glassmorphism, CSS Modules
- **Animations**: Framer Motion
- **3D Graphics**: Three.js
- **Icons**: Lucide React
- **Celebration Effects**: Canvas Confetti

---

## 💻 Local Setup & Installation

1. **Navigate to the portfolio directory**:
   ```bash
   cd portfolio
   ```

2. **Install project dependencies**:
   ```bash
   npm install
   ```

3. **Start the local development server**:
   ```bash
   npm run dev
   ```

4. **Build for production deployment**:
   ```bash
   npm run build
   ```
